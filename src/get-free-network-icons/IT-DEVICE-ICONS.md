## üöÄ Best Practices

### For IT Asset Management
1. **Use consistent icons** - Same icon = same device type
2. **Add labels** - Include text labels below icons
3. **Color code by status** - Green=active, Red=issues, Gray=inactive
4. **Show quantities** - "Desktop PC (x150)"

### For Network Diagrams
1. **Layer appropriately** - Users at top, infrastructure below
2. **Group logically** - All PCs together, all servers together
3. **Show relationships** - Use lines/arrows for connections
4. **Include legend** - Explain what each icon represents

### For Documentation
1. **Size consistently** - Pick one size (48px, 64px, etc.)
2. **Align properly** - Keep icons aligned in diagrams
3. **Use white space** - Don't overcrowd
4. **Label clearly** - Every icon should have a text label
